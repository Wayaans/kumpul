# editor-bars

Adds themed separator-line widgets above and below the pi editor without replacing the built-in footer.

## Behavior

- Hides the built-in editor's top and bottom border lines.
- Above editor: full-width separator line with footer-style token/context stats on the right with separators.
- Below editor: full-width separator line with provider, model, and thinking level on the left.

Both lines use the same thinking-level color as the editor border.

## Commands and tools

None. The widgets install automatically on session start.
